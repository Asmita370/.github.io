# Resume Website

## Objective
To create a simple resume website using HTML and host it using GitHub Pages.

## Description
This project is developed as part of the Application Development Lab.
It is a basic personal resume website created using HTML.

The website contains basic details such as:
- Name
- Education
- Skills
- Contact information

## Technologies Used
- HTML
- GitHub Pages

## How to View the Website
The resume website can be accessed using the following link:

https://asmita370.github.io

## Conclusion
This experiment helped in understanding basic HTML structure
and how to host a static website using GitHub Pages.
